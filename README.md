## 랜덤번호 뽑기 프로젝트 설명
#### 숫자를 입력한다. 이 값은 범위를 지정하는 것이며, 뽑을 때 마다 1부터 N 까지 범위 내 한 개의 번호를 화면에 보여진다. 마지막으로 여러 개의 뽑은 번호를 리스트로 확인도 가능하다
