## RandomNumber 프로젝트
### 프로젝트 설명
#### 번호의 범위를 설정하면 범위 내의 랜덤 값을 TextView에 표시하고 리스트로 보여진다.

### 프로젝트 목적
#### MVVM 패턴에서 ViewModel의 역할, 클래스 내에서 LiveData와 MutableLiveData 활용방법
#### MVVM 패턴의 이해
#### MVVM 패턴과 RecyclerView의 연결과 구조 이해
